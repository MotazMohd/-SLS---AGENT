## Design System

All screens follow the SALIS AUTO design system:

- **Colors**: Grayscale palette (gray-50 to gray-900, salis-black, salis-gray-dark)
- **Typography**: Montserrat for headings/numbers, Poppins for body text
- **Components**: shadcn/ui with Radix UI primitives
- **Icons**: Lucide React icon library
- **Theme**: Dark mode with class-based toggle
- **Responsive**: Mobile-first responsive design
- **Accessibility**: WCAG 2.1 AA compliant
