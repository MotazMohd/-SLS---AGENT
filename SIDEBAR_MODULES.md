The sidebar navigation is filtered based on user roles:

| **Admin / Super Admin** | All groups visible |

## Sidebar Features

### Resizable Width
- **Minimum Width**: 200px
- **Maximum Width**: 400px
- **Default Width**: 280px
- **Persistence**: Width saved to localStorage

### Collapsible Groups
- All groups are expanded by default
- Click group header to collapse/expand
- Smooth animation transitions

### Mobile Responsive
- Hamburger menu on mobile devices
- Full-screen overlay navigation
- Touch-friendly tap targets
