# SALIS AUTO - Keyboard Shortcuts & Quick Actions Guide

## Overview

SALIS AUTO provides a powerful Quick Actions system accessible via keyboard shortcuts, enabling rapid navigation and task execution across the platform's 141+ modules.

---

## Global Keyboard Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + K` (Windows/Linux) | Open Quick Actions | Opens the Quick Actions modal for rapid navigation |
| `Cmd + K` (macOS) | Open Quick Actions | Opens the Quick Actions modal for rapid navigation |
| `Escape` | Close Modal | Closes any open modal or dialog |
| `Enter` | Select Action | Executes the selected action in Quick Actions |
| `Arrow Up/Down` | Navigate Actions | Navigate through Quick Actions list |

## Using Quick Actions

### Opening Quick Actions
1. Press `Ctrl + K` (Windows/Linux) or `Cmd + K` (macOS)
2. The Quick Actions modal appears with a search field

### Searching
- Type any keyword to filter actions
- Search matches action names, descriptions, and keywords
- Examples:
  - Type "smart" to see all AI-powered features
  - Type "customer" to see customer-related actions
  - Type "portal" to see Customer Portal features

### Executing Actions
1. Use arrow keys to navigate the filtered list
2. Press `Enter` to execute the selected action
3. Or click directly on any action

### Closing
- Press `Escape` to close without action
- Click outside the modal to close

## Keyboard Navigation Best Practices

1. **Speed Tip**: Press `Ctrl+K`, type 2-3 letters, press `Enter` for instant navigation
2. **Smart Features**: Type "smart" to quickly access all AI-powered features
3. **Portal Access**: Type "portal" or "client" for customer self-service features
4. **Inventory**: Type "parts" or "inventory" for stock-related actions
