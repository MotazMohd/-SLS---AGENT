## Testing Workflows

### Complete Service Workflow
1. **Service Advisor**: Create appointment
2. **Receptionist**: Check in customer
3. **Service Manager**: Create job card
4. **Technician**: Execute service
5. **QC Inspector**: Inspect work
6. **Service Advisor**: Create invoice
7. **Accountant**: Process payment

### Parts Ordering Workflow
1. **Technician**: Request parts
2. **Parts Manager**: Check inventory
3. **Parts Manager**: Create purchase order
4. **Warehouse Manager**: Receive parts
5. **Parts Manager**: Update inventory
6. **Technician**: Use parts on job

### Customer Service Workflow
1. **Call Center Agent**: Receive call
2. **CSR**: Schedule appointment
3. **Service Advisor**: Provide estimate
4. **Customer**: Approve service
5. **Service Team**: Execute work
6. **Service Advisor**: Follow up
